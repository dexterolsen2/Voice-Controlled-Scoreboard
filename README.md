Basically, you have to say:

Plus or Minus
   |
   |
   |
"____ _ Scoreboard ____"
      ^              ^
      ^              ^
      ^              ^
 1, 2, or 3     Home or Away

EXAMPLE: "Plus 2 Scoreboard Away"
EXAMPLE 2: "Minus 1 Scoreboard Home"
EXAMPLE 3: "Minus 3 Scoreboard Away

That's basically it!
